6.1
- Nếu người dùng chọn Cancel, phương thức showConfirmDialog trả về JOptionPane.CANCEL_OPTION (giá trị 2). Panel "You've choosen No" hiện ra
- Để tùy biến nút lựa chọn, dùng JOptionPane.showOptionDialog và truyền vào mảng Object[] options (ví dụ: {"Yes","No"} hoặc {"I do","I don't"}).
import javax.swing.JOptionPane;
Ví dụ:
	public class hehe {
    		public static void main(String[] args) {
        		Object[] options = { "Yes", "No" }; // Custom options
        		int option = JOptionPane.showOptionDialog(null,
                		"Do you want to change to the first class ticket ? ",
                		"Status",
                		JOptionPane.YES_NO_OPTION,
                		JOptionPane.QUESTION_MESSAGE,
                		null,
                		options,
                		options[0]);

        		JOptionPane.showMessageDialog(null,
                		"You've chosen" + (option == JOptionPane.YES_OPTION ? " Yes" : " No"));

        		System.exit(0);
    		}
	}
